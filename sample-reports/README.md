# Sample Reports

These are example outputs from Proscan scans using fabricated project data. They show the structure, content, and formatting of reports you can generate from the platform.

All data in these samples is fictional. Project names, file paths, finding counts, and scores are made up for illustration purposes.

| Report | Description |
|--------|-------------|
| [SAST Scan Results](sast-scan-results.json) | JSON output from a SAST scan showing findings with severity, CWE, location, and fix suggestions |
| [PCI DSS Compliance](pci-dss-compliance.json) | Compliance report mapping scan findings to PCI DSS 4.0 controls |
| [Executive Summary](executive-summary.json) | High-level security posture overview for non-technical stakeholders |
| [SARIF Output](sarif-output.json) | SARIF 2.1.0 output compatible with GitHub Code Scanning and GitLab Security Dashboard |
| [CycloneDX SBOM](cyclonedx-sbom.json) | Software Bill of Materials in CycloneDX 1.5 format |
