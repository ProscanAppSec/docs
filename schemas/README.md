# Report Schemas

JSON schemas describing the structure of Proscan report outputs. These are useful for building integrations that consume Proscan reports programmatically.

| Schema | Description |
|--------|-------------|
| [scan-results.json](scan-results.json) | SAST/DAST/SCA scan results output |
| [compliance-report.json](compliance-report.json) | Framework-specific compliance report |
| [executive-summary.json](executive-summary.json) | Executive summary report |
